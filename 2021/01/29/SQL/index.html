# SQL

SQL 是数据库查询的终极模式，以代码的形式输入参数，是目前所有数据库的通用语言
常用的 SQL 查询有操作查询

# 操作查询

操作查询有四的分别为：生成表查询，追加查询，更新查询，删除查询

# SQL 的基本结构

select <字段名>
from <表名>
where <条件表达式>
group by <分组条件>
order by <排序字段> ASC||DESC

order 为排序关键字，默认升序，升序为 ASC , 降序为 DESC
